# README

File | Purpose
:--- | :------
`profile.sh` | Main
`profile_alias.sh` | Alias customization
`profile_path.sh` | PATH customization
`profile_prompt.sh` | Prompt customization

## Usage

Add `source "<PATH_TO_THIS_DIR>/profile.sh"` to your bash profile.
